產品設計 Product Designer 1位

上恩資訊 (UPN) 的使命是利用網路的科技，幫助家庭解決生活中遇到的種種問題，陪伴家庭與孩子快樂地成長。
在 UPN, product designer 跟 app developer / web designer 同樣處在我們產品開發的核心，他們除了喜歡設計、熱愛設計，更想挑戰自己：

  1. 如何更深入地思考如何透過產品解決消費者的問題
  2. 如何打造實用又美觀、在數據上也有所成長的產品
  3. 如何透過同理心，打造更暖心的服務
  4. 不只服務台灣的家庭，希望自己以打造世界一流的網路產品為目標

如果你也是充滿產品設計的靈魂，渴望透過產品來改善人們的生活，歡迎跟我們一起踏上這條偉大的航道！

Why you should consider UPN 為什麼你應該考慮上恩
  • 公司裡面臥虎藏龍，有許多 一 (ㄍ ㄨ ㄟ ) 流 (ㄇ ㄠ ˊ) 的 developers / designers
  • 公司就在古亭捷運站附近，交通方便 。
  • 未出校門就開始累積實戰的經驗與能力
  • 體驗新創公司的熱情與快速執行力，對之後的工作能力大大加分

工作內容：
  1. co-work with PM / Developers, 了解團隊優先順序並進行 app / web 的產品設計
  2. 吸收新知並動手實作，了解 app design / web design 的最佳實踐
  3. 收集與分析產品相關數據，了解產品該如何改進
  4. 透過分享與交流，讓 team members 也能有更多設計的思維

徵求條件
  1. 有過 1 年以上 UI/UX 設計的經驗 (app or web or 互動設計, 需附作品集並挑一個最滿意的作品加以說明)
  2. 喜歡 design , 而且覺得好的 design 不是只有考慮 "怎麼把這東西弄得美美的", 也應該考慮 "到底我們應該要 build 什麼東西"
  3. 有良好的溝通能力，能夠清楚溝通自己的設計理念，但同時又有接受不同意見的雅量
  4. 想挑戰自我的能力，願意嘗試在快節奏的公司工作
  5. data-driven, 喜歡用數字來驗證工作的成效而不是只是憑感覺
  6. 喜歡團隊合作 more than 一人球隊
  7. (bonus) 熟悉 sketch，有與 develoeprs 協作的經驗
  8. 熟悉 Android / iOS 設計規範
