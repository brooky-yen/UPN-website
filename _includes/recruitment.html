Web Developer 2位

上恩資訊 (UPN) 的使命是利用網路的科技，幫助家庭解決生活中遇到的種種問題，陪伴家庭與孩子快樂地成長。
在 UPN, web developer 跟 app developer / product designer 同樣處在我們產品開發的核心，他們除了身懷打造 web application 的絕世武功，更想挑戰自己：
  • 以打造世界一流的網路產品為目標
  • 以解決消費者的問題為己任
  • 以客觀的數據為指引
  • 以體貼客戶的心為依歸
如果你也是身懷武功但又不滿於此，希望能接受進一步的挑戰，歡迎跟我們一起踏上這條偉大的航道！

Why you should apply
  • 公司裡面臥虎藏龍，有許多 一 (ㄍ ㄨ ㄟ ) 流 (ㄇ ㄠ ˊ) 的 developers / designers
  • 身邊充滿的熱情與快速執行力的夥伴，讓你每天上班都活力充沛！
  • 公司就在古亭捷運站附近，交通方便 。
  • 開放的討論氣氛，讓你的 idea 有發揮的空間

Responsibilities 工作內容
  1. co-work with PM / designers, 了解團隊優先順序並進行 web application 開發
  2. 吸收新知並動手實作，了解 web application 的最佳實踐
  3. 收集與分析產品相關數據，了解產品該如何改進

Requirements 徵求條件
  1. 有過 1 年以上開發 WEB 的經驗
  2. 喜歡寫 code ，覺得在透過網路解決人們生活的難題是一件很酷的事
  3. 想挑戰自我的能力，願意嘗試在快節奏的公司工作
  4. 具 programming 經驗，不用得過 hackthon 冠軍，但是至少有寫過一些程式
  5. data-driven, 喜歡用數字來驗證工作的成效而不是只是憑感覺
  6. 溝通能力佳, 喜歡團隊合作 more than 一人球隊
