Product Manager 電商產品經理 1位

上恩資訊 (UPN) 的使命是利用網路的科技，幫助家庭解決生活中遇到的種種問題，陪伴家庭與孩子快樂地成長。
在 UPN, Product manager 產品經理的任務是幫助上恩開發種種可以幫助家庭解決問題的產品，而 E-commerce PM 更專注於上恩資訊 E-commerce 相關的產品，包括但不限於媽咪愛的網站及 app。

產品經理是一個很有挑戰性的工作，從產品數字的分析、產品規劃、籌措資源、跨組的溝通與協調，只要是可以幫助產品成長的，可能都是這個工作的一部份。

優秀的產品經理對於一個產品的成功扮演一個很重要的角色。也因為如此，我們希望膽大心細、勇於嘗試的你/妳，跟我們一起踏上這條偉大的航道。
  • 公司裡面臥虎藏龍，有許多 一 (ㄍ ㄨ ㄟ ) 流 (ㄇ ㄠ ˊ) 的 developers / designers
  • 公司就在古亭捷運站附近，交通方便 。
  • 未出校門就開始累積實戰的經驗與能力
  • 體驗新創公司的熱情與快速執行力，對之後的工作能力大大加分

工作內容：
  1. 與同樣充滿熱情的一 (ㄍ ㄨ ㄟ ) 流 (ㄇ ㄠ ˊ) developers / designers 一起工作，協助團隊開發優秀的產品
  2. 跨組溝通協調，協助產品團隊與公司其他部門取得優先順序的共識
  3. 透過使用者的研究調查、或是數字的分析等種種工具，協助持續優化產品，提升使用者滿意度
  4. 協助規畫與實驗不同的產品特色，並透過數據化的分析協助團隊確認有效的攻擊點
  5. 研究一流的 EC 產品，打造一流的 EC 產品

徵求條件
  1. 擁有 2 年以上打造 web / app 產品的經驗
  2. 絕佳的溝通能力
  3. 分析思考，可以有組織地思考並找到問題的核心
  4. 對 web application / mobile app 背後的運作架構有一定的認識，能夠知道不同解決方案背後的成本
  5. 對電子商務的運作有一定的認作，對於如何取得優質的流量、優化轉換率、提升顧客黏性及回購率有一定的經驗
  6. 對規劃簡單易用的 UI 以解決使用者的問題充滿熱忱，並有相關的實作經驗
  7. 不斷想要挑戰自我，願意嘗試在快節奏的公司工作
