職缺

Senior Web Developer 1位

上恩資訊 (UPN) 的使命是利用網路的科技，幫助家庭解決生活中遇到的種種問題，陪伴家庭與孩子快樂地成長。
在 UPN，web developer 跟 app developer / product designer 同樣處在我們產品開發的核心，他們除了身懷打造 web application 的絕世武功，更想挑戰自己：

  1. 以打造世界一流的網路產品為目標
  2. 以解決消費者的問題為己任
  3. 以客觀的數據為指引
  4. 以體貼客戶的心為依歸

如果你也是身懷武功但又不滿於此，希望能接受進一步的挑戰，歡迎跟我們一起踏上這條偉大的航道！

Responsibilities 工作內容：
  1. 協助團隊建立適合的系統架構去解決遇到的問題

  3. 想挑戰自我的能力，願意嘗試在快節奏的公司工作
  4. 溝通能力佳，喜歡團隊合作更勝一人球隊
  5. (bonus) 有過帶人的經驗
